# ⚖️ 판례 검색 수집기 (lawcase_search)

대법원 **사법정보공개포털 · 종합법률정보 「판례」**에서 판례를 검색해
**법원·사건번호·사건명·선고일자·판시사항·판결요지(전문)** 를 모아 CSV로 저장하는 도구입니다.

- 🌐 **웹 버전 (GitHub Pages):** <https://jonghhhh.github.io/lawcase_search/>
  검색어·필터를 고르고 버튼 한 번으로 CSV 다운로드.
- 🐍 **파이썬 버전 (`lawcase_search.py`):** 대량 수집용. CORS 프록시 불필요, 더 빠르고 안정적.

---

## 데이터 출처와 수집 방식

데이터는 [사법정보공개포털 종합법률정보 판례](https://portal.scourt.go.kr/pgp/index.on?m=PGP1011M01&l=N&c=900)에서 가져옵니다.

처음에는 **Playwright**로 브라우저를 자동 조작해 화면을 긁어왔지만, 이 포털은 화면이
자바스크립트로 그려지는 **SPA(WebSquare)** 구조라 요소 위치(셀렉터)가 자주 바뀌어 수집이 쉽게 깨졌습니다.
그래서 화면을 거치지 않고 포털이 내부적으로 호출하는 **검색 API를 직접 호출**하도록 바꿨습니다. (로그인 불필요)

| 단계 | 엔드포인트 | 설명 |
|---|---|---|
| 목록 검색 | `POST /pgp/pgp1011/selectJdcpctSrchRsltLst.on` | 검색어·필터·페이지 → 판례 목록 + 총 건수. 요지는 약 260자 미리보기로 잘려서 옴 |
| 요지 전문 | `POST /pgp/pgp1011/selectJdcpctSumrInf.on` | 일련번호로 호출 → 판시사항(`01`)·판결요지(`02`) 전문을 항목별로 받아 이어 붙임 |

### 웹 버전이 CORS 프록시를 쓰는 이유
브라우저는 보안 정책(CORS)상 다른 도메인(`scourt.go.kr`)의 API를 직접 호출할 수 없습니다.
그래서 웹 버전은 공개 CORS 프록시(`https://proxy.cors.sh/`)를 거쳐 요청을 전달합니다.
프록시가 막히거나 느릴 수 있으니, 그럴 때는 화면의 **고급 설정**에서 다른 프록시로 바꾸거나
**파이썬 버전**(프록시 불필요)을 사용하세요.

## 검색 옵션 / 필터 코드

API 분석으로 확인한 필터 값입니다. (여러 개 선택 시 OR 조건, 미선택 시 ‘전체’)

| 구분 | API 필드 | 값 |
|---|---|---|
| 법원종류 | `jisJdcpcInstnDvsCdGrp` | 대법원 `01` · 고등법원 `03` · 하급심 `04` |
| 판례등급 | `grpJdcpctGrCdGrp` | 전원합의체 `01` · 간행판결 `02` · 미간행판결 `03` · 변경·폐기 `04` |
| 사건종류 | `jdcpctCdcsCdGrp` | 민사 `01` · 형사 `02` · 가사 `03` · 특허 `04` · 조세 `05` · 행정 `06` |
| 선고구분 | `adjdTypCdGrp` | 판결 `01` · 결정 `02` |

> 참고: 포털 API에서 **변경**과 **폐기**는 같은 그룹 코드(`04`)로 묶여 내려오므로 ‘변경·폐기’로 함께 제공합니다.

## 파이썬 버전 사용법

```bash
pip install -r requirements.txt
```

`lawcase_search.py` 상단의 설정만 바꾸고 실행합니다.

```python
KEYWORD = "산업재해"        # 검색어
MAX_RECORDS = None          # 최대 수집 건수 (None = 전체)
FETCH_FULL_TEXT = True      # 요지 전문 포함 여부

COURT_FILTER      = []      # "대법원", "고등법원", "하급심"
GRADE_FILTER      = []      # "전원합의체", "간행판결", "미간행판결", "변경·폐기"
CASE_TYPE_FILTER  = []      # "민사", "형사", "가사", "특허", "조세", "행정"
DECISION_FILTER   = []      # "판결", "결정"
```

```bash
python lawcase_search.py
```

실행하면 `scourt_<검색어>_요지.xlsx` / `.csv` 가 만들어집니다.

본 도구는 비상업적 **연구·학습 목적**의 보조 도구입니다.
판례 데이터의 저작권 및 이용 조건은 사법정보공개포털 정책을 따르며,
