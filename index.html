⚖️ 판례 검색 수집기

대법원 사법정보공개포털 · 종합법률정보에서 판례를 검색해 요지 전문까지 모아 CSV로 내려받습니다.

이 도구는 무엇인가요?

대한민국 법원이 운영하는 사법정보공개포털 종합법률정보 「판례」 검색 결과를 자동으로 수집하는 웹 도구입니다. 검색어와 필터(법원종류·판례등급·사건종류·선고구분)를 지정하면, 각 판례의 법원·사건번호·사건명·선고일자·판시사항·판결요지(전문)를 모아 엑셀에서 바로 열리는 CSV 파일로 저장합니다.

어떻게 데이터를 가져오나요?

처음에는 Playwright로 브라우저를 띄워 화면을 클릭·스크롤하며 긁어오는 방식을 썼습니다. 그런데 이 포털은 화면이 자바스크립트로 그려지는 SPA(WebSquare) 구조라 버튼·목록의 위치(셀렉터)가 자주 바뀌어 수집이 쉽게 깨졌습니다.

그래서 화면을 거치지 않고, 포털이 내부적으로 호출하는 검색 API를 직접 호출하도록 바꿨습니다. 훨씬 빠르고 안정적이며, 로그인도 필요 없습니다.

  • 목록 검색 — POST /pgp/pgp1011/selectJdcpctSrchRsltLst.on
    검색어·필터·페이지를 보내면 판례 목록과 총 건수를 돌려줍니다. (요지는 약 260자 미리보기로 잘려 옵니다)
  • 요지 전문 — POST /pgp/pgp1011/selectJdcpctSumrInf.on
    판례 일련번호로 호출해 판시사항(코드 01)·판결요지(코드 02) 전문을 항목별로 받아 이어 붙입니다. '요지 전문 포함'을 켜면 판례 1건당 이 호출을 추가로 합니다.

CORS 프록시가 필요한 이유

브라우저는 보안 정책(CORS)상 다른 도메인(scourt.go.kr)의 API를 직접 부를 수 없습니다. 그래서 이 페이지는 공개 CORS 프록시(proxy.cors.sh)를 거쳐 요청을 전달합니다. 프록시가 막히거나 느릴 수 있으니, 그럴 때는 아래 ‘고급 설정’에서 다른 프록시 주소로 바꾸거나, 대량 수집은 저장소의 파이썬 스크립트 lawcase_search.py(프록시 불필요)를 사용하세요.

필터 코드 (포털 API 분석으로 확인)

  • 법원종류 대법원·고등법원·하급심
  • 판례등급 전원합의체·간행판결·미간행판결·변경·폐기 (API에서 ‘변경’과 ‘폐기’는 같은 그룹으로 묶여 ‘변경·폐기’로 제공)
  • 사건종류 민사·형사·가사·특허·조세·행정
  • 선고구분 판결·결정
